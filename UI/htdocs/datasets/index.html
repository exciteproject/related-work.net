Arxiv

    This is a mirror of the meta of preprints at Arxiv, as provided by the OAI API in oai_dc format. Downloading full dataset from the API is a very slow. We think that git provides a much nicer way to share data sets of limited size (<1Gb) like this one.

  • Arxiv metadata mirror

    • Using latex source we matched all extracted references to our database. This file contains all matched references. The format is one JSON per line, and every line contains three fields: [meta_id_source, ref_text, meta_id_target] Number of entries: 39.069.913

  • Arxiv extracted references

    • Every downloaded arxiv PDF has been hashed with sha1, md5 and sha256. File format is one JSON per line, and every line contains 3 hashes and an arxiv id of the paper.

    This repository contains manually inspected datasets for evaluating the different steps during the reference extraction process. All datasets consist of research papers in German and come from the SSOAR repository.
